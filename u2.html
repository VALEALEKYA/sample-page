1. Bivariate Foundations: Correlation & Scatter Plots

Before executing predictions, spatial models must establish patterns. Correlation evaluates the scale strength and linear vector orientation between continuous feature metrics.

  • Scatter Plots: Two-dimensional feature coordinates mapped instantly across space. Clusters mapping along an ascending vector express positive attributes, whereas descending alignments confirm strict reverse relationships.
Pearson's Correlation Coefficient (r) Computational Formula: r = [ n(∑XY) - (∑X)(∑Y) ] / √[ {n∑X² - (∑X)²} * {n∑Y² - (∑Y)²} ]

Boundary Conditions: The 'r' configuration restricts strictly between -1.0 and +1.0. A score of absolute 0 indicates complete non-linear variance distribution.

2. Optimization Framework: Linear Regression & Least Squares

Linear Regression builds downstream systems over basic correlation by projecting data onto a continuous trend line equation modeled as: Y = mX + c.

📐 The Least Squares Criterion

The mathematical mechanism fits the optimized vector line by minimizing the continuous sum of squared residuals (differences between spatial observations and predicted track coordinates).

3. Quality Assessment & Error Metrics

🎯 Standard Error of Estimate

Measures precise residual standard deviation around your calculated line. It reveals the exact dispersion metric of expected system prediction errors.

📊 Decoupling Coefficient of Determination (R²)

R² presents the exact proportion of variance within the output variable explained by the input data model features. An R² metric of 0.85 validates that 85% of downstream structural changes are completely captured by your input feature sets.

4. Multi-Variable Extensions & System Mean Reversion

Multiple Regression Equations: Systems projecting variables by combining several input parameters over a multidimensional plane: Y = β₀ + β₁X₁ + β₂X₂ + ... + βₙXₙ.

Regression Towards the Mean: A fundamental statistical phenomenon where extreme initial sample runs natively pull back closer toward standard average baselines on subsequent measurement tracks.

Q1. Apply Linear Regression for a sample dataset and calculate the R² Value.

Step-by-Step Analytical Process:

Given Sample Set: X = [1, 2, 3, 4, 5], Y = [3, 4, 2, 4, 5].

1. Compute Means: X̄ = 3, Ȳ = 3.6
2. Calculate Slope (m): Sum((X - X̄)(Y - Ȳ)) / Sum((X - X̄)²) = 3 / 10 = 0.3
3. Calculate Intercept (c): Ȳ - mX̄ = 3.6 - (0.3 * 3) = 2.7
4. Model Equation: Y_pred = 2.7 + 0.3X

Using predicted steps, calculate sum of squared residuals (SS_res) and total variance (SS_tot) to establish R² ≈ 0.1735.

import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.metrics import r2_score

X = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)
Y = np.array([3, 4, 2, 4, 5])
model = LinearRegression().fit(X, Y)
print("Calculated R2 Score:", r2_score(Y, model.predict(X)))
